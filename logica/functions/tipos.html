Tipos de Funções em JavaScript

1. Função Declarada

São nomeadas e carregadas pelo interpretador antes da execução do código.

function saudacao() {
  return "Olá!";
}
console.log(saudacao());

2. Função Expressão (Anônima)

Função sem nome atribuída a uma variável. Não sofre hoisting.

const soma = function(a, b) {
  return a + b;
};
console.log(soma(3, 4));

3. Arrow Function

Função de sintaxe curta, sem seu próprio this.

const dobro = x => x * 2;
console.log(dobro(5));

4. IIFE (Função Autoexecutável)

Executa assim que é definida. Útil para encapsular código.

(function() {
  console.log("Executada automaticamente!");
})();

5. Callback

Função passada como argumento para outra função.

function mostrar(msg) {
  console.log("Callback:", msg);
}
function processar(valor, funcaoCallback) {
  funcaoCallback(valor);
}
processar("Exemplo", mostrar);
